# Feature
- 사용자
  - ID, PW 회원가입
  - 일반 로그인, 소셜 로그인(카카오)
- 게시판
  - 게시글 및 댓글 CRUD
  - 키워드 검색 (제목, 내용, 작성자)
  - 게시판 목록 페이징
  - 게시판 목록 댓글 개수 보여주기

 # Tech 
 - Java 11/ Spring Boot v2.7.14
 - Gradle
 - MySQL (DB)
